# EvaOne.AI

EvaOne.AI is a local-first executive command app for iPhone built with Expo and React Native.

## What this starter includes

- local SQLite persistence with `expo-sqlite`
- Command Center for turning inputs into tasks, memory, or ops logs
- task management with done / reopen actions
- memory capture for notes worth keeping
- operations console for session flags and activity history
- dark premium mobile-first interface
- no fake app control and no fake web powers

## Start it

```bash
npm install
npx expo start
```

## Why this starter uses Expo SDK 54

This project is pinned for physical iPhone testing in Expo Go during the current SDK 55 transition window.
